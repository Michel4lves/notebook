Comandos gerais e dicas:

type()

Verifica o tipo do objeto.

type('Olá Mundo')

Resultado:

<class 'str'>

Alguns tipos de objetos python:

Tipo Descrição
int número inteiro
float número real
list lista
tuple tupla
dict dicionário

id()

Retorna o ID numérico do objeto.

id('Michel')

Resultado:

140160717856624

Comparações são diretas

n = 5
1 < n and n < 7
1 < n < 7

Resultado:

True
True

Pode-se adicionar strings e listas a operações matemáticas

'a' * 4
[1, 3] * 3

Resultado:

aaaa
[1, 3, 1, 3, 1, 3]
